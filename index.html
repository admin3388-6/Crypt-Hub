🛠️ How The Tools Work

1. Keystore Generator: Uses native Java keytool integrated with Bouncy Castle libraries to generate industry-standard and specialized cryptographic keys (APK/AAB signing). It automatically extracts your unique SHA fingerprints.

2. Quantum File Vault: Uses a military-grade Cascade Encryption architecture. Your password is hashed using Argon2id (GPU/ASIC resistant). The file is then encrypted using AES-256-GCM (Layer 1) and wrapped again in ChaCha20-Poly1305 (Layer 2). This makes brute-forcing or quantum-computing attacks practically impossible.
